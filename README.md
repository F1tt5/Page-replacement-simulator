# Симулятор алгоритмов замещения страниц в ОС

Проект реализует симулятор работы виртуальной памяти с различными алгоритмами замещения страниц. Разработан как выпускная квалификационная работа бакалавра.

# Быстрый старт:

### Требования
- Windows 10/11
- Visual Studio 2022 (или Visual C++ Redistributable)
- 64-разрядная система

### Запуск
1. Скачайте последнюю версию из раздела [Releases](https://github.com/ваш-логин/репозиторий/releases)
2. Создать файл с исходными данными симуляции (примеры представлены в x64/Release)
3. Выполните в командной строке:
   ```cmd
   VKR.exe < Название_исходного_файла.txt
Запуск через cmd:
   cmd /c VKR.exe < EVMAging.txt

### Результаты работы
Программа создает два типа файлов:

- results_*.txt - основная статистика
- Journal_*.txt - лог выполнения (при соответствующем параметре в исходном файле)
